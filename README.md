# Franklin-s-Lectures-Website


Project Overview

This project is a Flask-based web application that simulates a learning platform where students can browse B.Tech departments, explore semester structures, and view available courses. The system demonstrates basic backend web development concepts such as routing, template rendering, dynamic URL parameters, and form handling.
The application uses Python Flask for the backend and Jinja templating to render dynamic HTML pages.
Technologies Used

Backend:
Python
Flask

Frontend:
HTML
CSS
Jinja2 Template Engine

Other Tools:
Flask Routing
Flash Messaging
Dynamic URL generation using url_for()
